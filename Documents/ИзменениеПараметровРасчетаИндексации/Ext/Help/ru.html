Документ «Изменение параметров расчета индексации»

 

Документ предназначен и для установки базовых периодов, которые участвую в алгоритме расчета индексации заработной платы. Базовым периодом является период, в котором производится повышения заработной платы.

Документ может быть введен на основании документов изменения зарплаты сотрудников ("Ввод сведений о плановых начислениях работников организаций", "Кадровое перемещение организации")или введен самостоятельно.

В документе также может быть рассчитана сумма фиксированной индексации сотрудников по результатам повышения заработной платы.

1. Для сотрудников, которым производится повышение заработной платы, в месяце такого повышения вводится документ "Изменение параметров расчета индексации".

2. В документ производится расчет начислений таких сотрудников до повышения заработной платы с учетом индексации (табличная часть "Начисления до") и расчет начислений после повышения без учета индексации (табличная часть "Начисления после"). Если сумма "начислений до" превышает сумму "начислений после" - разница фиксируется в качестве суммы фиксированной индексации сотрудников.
3. При последующем повышении заработной платы вновь оформляется документ "Изменение параметров расчета индексации" в котором
      • если сумма повышения превышает сумму индексации с учетом фиксированной индексации - фиксированная индексация для сотрудника отменяется;
      • если сумма повышения превышает сумму индексации без учета фиксированной индексации - фиксированная индексация сотрудника не изменяется;
      • если сумма повышения меньше суммы индексации без учета фиксированной индексации - фиксированная индексация сотрудника увеличивается на сумму такой разницы.

4. При необходимости - сумма фиксированной индексации сотрудника может быть изменена или установлена самостоятельно.

 

Если установить базовый период для сотрудников необходимо без учета изменений заработной платы и расчета суммы фиксированной индексации - в документе устанавливается флаг "Установить базовый период без учета начислений".
