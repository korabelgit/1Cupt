Документ "Запись книги приобретения"

Документ предназначен для регистрации первичных документов, являющихся основаниями для внесения записей в реестр налоговых накладных. Каждый документ «Запись книги приобретения» соответствует одной строке в реестре.

Начиная с 01.01.2015, документ предназначен для регистрации входящих документов, необходимых для начисления налогового кредита и заполнения раздела II Декларации по НДС.

Таможенные декларации по импорту регистрируются документами «ГТД по импорту». «Запись книги приобретения» по таким операциям не оформляется.
