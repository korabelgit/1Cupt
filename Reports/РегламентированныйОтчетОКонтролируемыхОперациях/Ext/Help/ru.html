Отчет "О контролируемых операциях"

Отчет предназначен для заполнения и формирования регламентированного "Отчета о контролируемых операциях" в соответствии с приказом Миндоходов № 669 от 11.11.2013 г.

Заполнять отчет нужно в следующей последовательности:

1. Сведения о связанности контролируемых лиц. Для каждой организации предприятия вносится перечень контрагентов, операции с которыми являются контролируемыми в соответствии с пп. 39.2.1.1 и 39.2.1.2 НКУ: связанные лица-нерезиденты; связанные лица-резиденты определенных категорий; нерезиденты из стран с низкой ставкой налога на прибыль. При заполнении регистра сведений "Связанность лиц контролируемых операций" обязательно сразу заполнять "Код операций, относимых к контролируемым" и "Код связанности лиц". Заполненность кодов является обязательным условием для того, чтобы контрагенты считались связанными с организацией, и для автоматического заполнения документа Сведения о контролируемых операциях.

2. Дополнительные настройки, используемые при регистрации сведений о контролируемых операциях. Вносятся сведения
- Предметы контролируемых операций: предметом операции может быть Номенклатура, ОС, Объект строительства или НМА;
- Договоры участников контролируемых операций: информация о договорах (контрактах) с контрагентами, в рамках которых осуществлялись контролируемые операции;
- Параметры определения обычных цен контролируемых операций: коды оснований, методов и источников для определения обычных цен, указываемые в отчете; параметры могут быть указаны как для предприятия (организации) в целом, так и для отдельных предметов операций;
- Обычные цены контролируемых операций: указываются значения обычных цен предметов операций.

3. Документы, регистрирующие сведения о контролируемых операциях. Формируется документ Сведения о контролируемых операциях. В документе содержится информация обо всех контролируемых операциях одной организации в течение года. Документ может быть заполнен автоматически по кнопке "Заполнить за период". При автоматическом заполнении в документ включаются все операции с контрагентами, включенными в регистр сведений Связанность лиц контролируемых операций (см. п. 1). Для автоматического заполнения всех реквизитов отчета предварительно должны быть заполнены дополнительные настройки (см. п. 2).
При автоматическом заполнении не учитывается инфрормация о возвратах (из документов "Возврат товаров от покупателя" и "Возврат товаров поставщику"

4. Контрагенты, включаемые в отчет о контролируемых операциях. Указываются только контрагенты, операции с которыми являются контролируемыми и должны попадать в "Отчет о контролируемых операциях". Регистр сведений может быть заполнен автоматически по кнопке "Заполнить список", при этом в список попадают контрагенты, указанные в документе Сведения о контролируемых операциях, сумма операций с которыми в течение года составляет не менее 50 млн. грн.

ВАЖНО! В настоящее время не определен однозначно состав операций, сумма которых должна составлять не менее 50 млн. грн. При автоматическом заполнении документа "Сведения о контролируемых операциях" и регистра сведений "Контрагенты, включаемые в отчет о контролируемых операциях" анализируются обороты по бухгалтерским счетам расчетов с контрагентами. В случае, если предприятие определяет сумму операций по другой методике, нужно:
- изменить "Сумму всех операций" на закладке "Суммы всех операций контрагентов" документа "Сведения о контролируемых операциях";
- вручную добавить контрагентов в регистр сведений "Контрагенты, включаемые в отчет о контролируемых операциях"; указать для них "№ приложения отчета" при помощи кнопки "Восстановить нумерацию приложений";
- вручную удалить из регистра сведений "Контрагенты, включаемые в отчет о контролируемых операциях" контрагентов, операции с которыми не должны попасть в Отчет о контролируемых операциях.

 

5. Проверить правильность заполнения информации о контролируемых операциях можно из формы отчета по кнопке "Проверить корректность данных". При нажатии кнопки осуществляется начальный контроль данных без формирования и передачи данных в 1С-Звит.

 

6. Формирование регламентированного отчета о контролируемых операциях с приложениями производится в подсистеме 1С-Звит. Экранной формы во встроенной системе регламентированных отчетов не предусмотрено. Заполнение отчетных форм в 1С-Звит происходит после нажатия на кнопку "Передать данные в 1С-Звит". При сохранении и подготовке к отправке отчетов в контролируемые органы, осуществляется контроль данных встроенными средствами системы 1С-Звит.
