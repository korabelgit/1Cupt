Сервис сдачи электронной отчетности (СЭО)

Сервис сдачи электронной отчетности - это совместная разработка фирмы «1С» и компании «БЭСТ Звит», представляющая собой программу «1С:Звіт», разработанную компанией «БЭСТ Звит» и средства интеграции данной программы с типовыми конфигурациями фирмы «1С».

О программе «1С:Звіт»

Продукт «1С:Звіт» позволяет отправлять отчетность предприятия в контролирующие органы, при этом пользователь получает квитанцию о доставке отчета, и результаты обработки документа. Принцип работы с программой «1С» обеспечивает автоматическое создание отчетов путем перенесения данных из «1С». Передача отчетов происходит при помощи электронной почты, с обязательным применением электронной цифровой подписи.

Сервис СЭО предоставляет возможность пользователям программных продуктов экономического назначения фирмы «1С» для Украины непосредственно из типовых конфигураций:

  • сформировать регламентированную отчетность в различные государственные органы;
  • проверить заполнение (при помощи встроенных камеральных проверок) и распечатать документы;
  • зашифровать и подписать электронной цифровой подписью;
  • отправить отчеты в контролирующие органы и получить квитанцию о доставке отчета по назначению;
  • сохранить на электронных носителях.

Пользователи смогут самостоятельно выбрать вариант использования подсистемы регламентированной отчетности: встроенная подсистема или сервис СЭО. При   разработке ставилась задача развить и расширить существующие сервисы формирования регламентированной отчетности. Тесная интеграция сервиса СЭО с типовыми конфигурациями позволяет передавать данные учетной системы без создания промежуточных файлов экспорта, а также получать расшифровки сумм показателей отчетов сервиса СЭО.

Установка «1С:Звіт»

В поставку новых продуктов включен дистрибутив однопользовательской версии программы «1С:Звіт». Существующие пользователи программных продуктов «1С» могут получить актуальную версию программы «1С:Звіт» на сайте www.1c-sed.com.ua без регистрации. Подробная инструкция по работе с программой включена в поставку дистрибутива, а также может быть получена на сайте www.1c-sed.com.ua.

Установка программы осуществляется с любого физического носителя: CD-диска, жесткого диска, флешки. Для установки программы достаточно запустить инсталляционный файл, и следовать подсказкам «Мастера». Программа может работать в локальном и сетевом вариантах. Локальный вариант программы предназначен для использования на одном рабочем месте, сетевой вариант - для одновременной работы нескольких пользователей с одной базой данных. Нужный вариант указывается в процессе инсталляции программы.

Настройка, первый запуск и активация программы «1С:Звіт»

  • После установки программы необходимо указать, какие организации будут использовать сервис СЭО. Для этого в форме соответствующего элемента справочника «Организации» необходимо указать дату начала использования «1С:Звіт». 

Важно! Перевод организации на сервис СЭО, не приводит к безвозвратной реструктуризации данных информационной базы. Это позволяет параллельное использование с встроенной подсистемой для постепенного перехода на сервис СЭО.

  • Далее требуется проверить и подтвердить настройки заполнения отчетов. Для этого необходимо открыть форму настроек через пункт меню Регламентированная отчетность - Настройка «1С:Звіт». При работе используются те же настройки, что и в встроенной подсистеме.
  • Необходимо выполнить запуск «1С:Звіт». Для этого необходимо выбрать пункт меню Регламентированная отчетность - Регламентированные отчеты «1С:Звіт». После запуска из «1С» в «1С:Звіт» автоматически будут переданы основные данные об организациях использующих «1С:Звіт».
  • Далее, в открывшемся окне «1С:Звіт», выбирается организация для которой будет формироваться регламентированная отчетность.
  • При первом запуске программа предложит ввести лицензионный код. Загрузить код лицензии можно автоматически с сайта www.1c-sed.com.ua, нажав на соответствующую копку. При генерации лицензионного кода учитываются регистрационные данные предприятия. Следовательно, активировать нужно каждую организацию отдельно.

Важно! В случае прекращения срока действия лицензионного кода «1С:Звіт» продолжает работать, однако функциональность будет ограничена.

  • Настройка завершена. В дальнейшем запуск «1С:Звіт» выполняется всегда из «1С» через пункт меню Регламентированная отчетность - Регламентированные отчеты «1С:Звіт».

Формирование регламентированной отчетности в программе «1С:Звіт»

Создание отчетности

Формировать регламентированную отчетность в различные государственные органы можно как самостоятельно, так и перенося данные автоматически из «1С». Для заполнения документов данными из «1С» достаточно нажать кнопку «Заполнить» в бланке созданного отчета. Список доступных отчетов для их заполнения можно получить в форме настройки «1С:Звіт» (для просмотра полного списка отчетов необходимо снять все фильтры).

Внимание! Выгрузка документов «Налоговая накладная» и «Приложение 2 к налоговой накладной» осуществляется из «1С». Для этого в форме документов добавлен пункт меню Выгрузка - Выгрузить в «1С:Звіт».

Проверка правильности заполнения документов

Обязательным этапом работы с отчетностью является проверка правильности заполнения документов. Проверка осуществляется при помощи встроенных камеральных проверок, и полностью соответствуют проверкам контролирующих органов. На этом этапе исключаются практически все возможные ошибки, что существенно увеличивает шанс сдачи отчета с первого раза.

Отправка отчетности в контролирующие органы

Непосредственно перед отправкой, документы шифруются и подписываются электронной цифровой подписью. В этот момент документ приобретает юридическую силу (оригинал документа в электронном виде соответствует документу с мокрой печатью).

Отправка отчетности в электронном виде с ЭЦП в контролирующие органы (ГНАУ, ПФУ, Госкомстат и другие) осуществляется через прямое соединение или посредством встроенного почтового клиента. Отчеты доставляются через сервер электронных документов, представляющий собой программно-аппаратный комплекс «1С-СЭД». Программа поддерживает одновременную отправку нескольких документов указанным получателям.

Программа принимает квитанцию о доставке отчета, а так же квитанции о результатах обработки документа. С момента получения второй квитанции, отчет считается сданной.

Дополнительные функции

В качестве дополнительного сервиса программа позволяет проверить состояние плательщика НДС из базы плательщиков ГНАУ и оценку рисков налоговой проверки.
