Загрузка входящих налоговых документов в формате XML из "1С:Звіт"

Обработка предназначена для пакетной загрузки входящих налоговых накладных и расчетов корректировки, полученных от контрагентов в электронном виде в формате XML через систему 1С:Звіт.

Порядок работы

  1. Таблица полученных документов в "1С:Звіт" заполняется при открытии формы или при нажатии на кнопку "Обновить список". В таблицу попадают документы, выписанные контрагентами за указанный период по выбранной организации. Возможно также установить отбор по определенному контрагенту.
  2. После получения списка входящих документов из "1С:Звіт" система автоматически определяет контрагента из информацинной базы по коду ИНН и коду ЕГРПО(ДРФО), полученными из системы "1С:Звіт".
  3. Если входящий документ в формате XML уже регистрировался в информационной базе 1С (создавался входящий налоговый документ), ссылка на такой документ будет показана в таблице. Также будет произведен анализ найденного документа из информационной базы 1С на соответствие ключевых полей: кодов контрагента, суммы НДС, номера и даты входящего налогового документа. Информация о найденных расхождениях будет показана в таблице.
  4. После загрузки списка документов из "1С:Звіт" в таблице будут установлены флажки "загружать XML?" для тех строк для которых не были обнаружены соответствующие документы в системе 1С либо в таких документах были обнаружены расхождения в ключевых полях. Если не удалось в информационной базе найти контрагента с соответствующими данным "1С:Звіт" кодами флажок загрузки также будет снят.
  5. Для строк таблицы, в которых не указан контрагент необходимо его указать вручную.
  6. Загрузка XML-образов входящих налоговых документов просихожит при нажатии на кнопку "Загрузить XML-Документы". Если в информационной базе 1С нет документа регистрации входящего налогового документа он будет создан. В таком документе будет заполнен только контрагент, номер и дата входящего налогового документа. Документы формируются текущей рабочей датой. После создания документов их можно открыть из списка. Загруженный входящий налоговый документ в XML-формате можно в дальнейшем открыть на просмотр из формы документа или формы данной обработки или в виде печатной формы, или в виде текстового файла XML, либо открыть документ в системе "1С:Звіт".
Замечание. Красным цветом выделяются документы,  для которых нарушен предельный срок регистрации в ЕРНН.
