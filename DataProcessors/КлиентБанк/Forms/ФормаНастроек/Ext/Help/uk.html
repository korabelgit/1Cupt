Настроювання параметрів обробки

Сторінка "Організація"

 

На цій сторінці необхідно вказати організацію і банківський рахунок організації за замовчуванням. Ці реквізити будуть автоматично підставлятися у відповідні поля при відкритті основної обробки.


 

Сторінка "Контрагент"

 

Якщо юридичне /фізична особа і контрагент при імпорті виписки не будуть автоматично знайдені по коду ЄДРПОУ/ДРФО, при встановленому прапорці "Автоматично створювати контрагентів і юридичних/фізичних осіб" будуть автоматично створені юридичне /фізична особа і контрагент у групах, зазначених нижче у відповідних полях.

Після закінчення імпорту користувач може перемістити створені елементи в інші групи.

Для різних напрямків платежу можна вказувати різні групи.

За замовчуванням обробкою створюються групи з назвою "Нові (клієнт-банк)".

 

Сторінка "Імпорт"

 

При встановленому прапорці "Автоматичний імпорт виписки при відкритті обробки" при відкритті обробки буде зроблений автоматичний імпорт виписки:

 

  • Організація, банківський рахунок будуть автоматично встановлені з настроювань обробки;
  • По банківському рахунку буде визначений формат обміну даними;
  • Будуть завантажені існуючі платежі - документи "Рядки банківських виписок";
  • Автоматично буде сформоване ім'я файлу імпорту (у залежності від параметрів формату обміну);
  • З файлу імпорту будуть завантажені нові вхідні і вихідні платежі.


При встановленому прапорці "Імпорт тільки вхідних платежів" при імпорті виписки будуть завантажуватися тільки вхідні платежі.

 

Сторінка "Експорт"

 

При встановленому прапорці "Автоматичне заповнення таблиці експорту при відкритті" при відкритті обробки таблиця експорту буде автоматично заповнена існуючими платіжними дорученнями.

 

Сторінка "Документи"

 

При встановленому прапорці "Проводити створені документи" при створенні документів - рядків виписок буде зроблена спроба провести створені документи.

 

При встановленому прапорці "Відкривати створені документи" після створення документів - рядків виписок створені документи будуть відкриті для редагування користувачем.

 

Сторінка "Системні"

 

У поле "Каталог продукту" необхідно вказати каталог, в якому був установлений продукт (у якому розміщені усі файли обробок, іnі-файли і т.д.)

 

 

Увага.  Якщо каталог вказано не правильно, обробка обміну не буде працювати!

 

У списку "Порядок сортування рядків у таблиці імпорту" необхідно настроїти порядок сортування рядків у таблиці імпорту:

  • Статус рядка виписки - сортування по статусу рядка таблиці імпорту (існуючий платіж; рядок із платежем, готовий до запису, рядок, у якого заповнені не всі реквізити);
  • Сума платежу приходу, витрати - сортування спочатку по напрямку платежу (прихід, витрата ), потім по сумі платежу;
  • По контрагентах, юридичним (фізичним ) особам - сортування по найменуванню контрагентів, юридичних(фізичних) осіб ;
  • По сумі - сортування по сумі платежу (незалежно від того, прихід це або видаток).

 

У полі вибору "Робота з поштою" необхідно вказати спосіб роботи з поштою при запиті файлів форматів обміну - вбудованими засобами 1С:Підприємства або за допомогою зв'язку з Mіcrosoft® Outlook®.
 
Спосіб "Через Mіcrosoft® Outlook®" варто вибирати, якщо ця програма встановлена і з яких-небудь причин внутрішні засоби 1С:Підприємства не дозволяють відправити поштове повідомлення з запитом файлів.

 

Сторінка "Формати"

 

На цій сторінці можна задати визначені настроювання для форматів обміну. Ці настроювання будуть прочитані в першу чергу, потім - з файлів настроювань (ini-файли). Для кожного МФО банку необхідно вказати ім'я формату (нове або вибрати з існуючих) і ім'я файлу обробки формату.
