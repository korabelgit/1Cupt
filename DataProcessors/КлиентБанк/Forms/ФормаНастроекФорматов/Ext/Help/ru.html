Выбор используемых форматов обмена

 

В этой форме необходимо выбрать используемые форматы обмена данными и при необходимости выслать запрос произврдителю продукта на требуемые файлы обработок обмена.

Уже используемые форматы обмена содержатся в таблице форматов. В колонке "Имя формата" этой таблицы отображается имя формата обмена, в колонке "Имя файла обработки формата" отображается соответствующее имя файла.

Если файла обработки формата не существует, строка таблицы будет отображаться красным цветом.

Имя формата обмена можно добавить как вручную, так и автоматически заполнить строки таблицы с помощью следующих кнопок:

  • "Заполнить по счетам" - при нажатии этой кнопки таблица форматов будет заполнена форматами, которые соответствуют всем банковским счетам всех организаций, которые существуют в информационной базе.
  • "Добавить по счету" - при нажатии этой кнопки необходимо выбрать организацию и банковский счет, после чего в таблицу форматов будет добавлена строка, которая соответствует формату обмена для выбранного банковского счета.

 

Для получения отсутствующих файлов форматов обмена необходимо выслать по электронной почте запрос производителю продукта.

Внимание. Перед высылкой запроса необходимо проверить список требуемых форматов обмена, так как стандартная поставка продукта позволяет получить бесплатно ограниченное число файлов формата обмена. Если число запрашиваемых форматов будет больше этого числа, будут высланы первые указанные в таблице файлы.

Для отправки запроса на файлы форматов необходимо нажать кнопку "Отправить запрос на файлы". Подробнее об отправке запроса рассказано в главе справки "Запрос файлов обработок форматов".

После получения ответного письма от производителя продукта необходимо скопировать все файлы-вложения письма в каталог продукта.

Для сохранения списка используемых форматов обмена необходимо нажать кнопку "Сохранить".
